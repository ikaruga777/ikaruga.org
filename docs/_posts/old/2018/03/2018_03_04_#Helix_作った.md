<h3>基盤とか</h3>    <ul>      <li>PCB + ステンレスプレートセット（５行版）</li>      <li>Pro Micro（互換）</li>      <li>ダイオード（チップ） 50個入り</li>      <li>LEDストリップ（WS2812B）</li>    </ul>    <p>      GBで頼みました。プレートをステンレスにしたので10000円ちょっとかかってますが、アクリルなら半額でした。    </p>    <h3>OLEDモジュール</h3>    <p>      普通にGBで頼み忘れてたのですが、同じようなものが<a href="http://amzn.asia/1IdkScI" target="_blank">Amazon</a>にあったので助かりました。    </p>    <h3>キーキャップ</h3>    <p>      <a href="https://talpkeyboard.stores.jp/" target="_blank">TALP KEYBOARD</a>さんのブランクキーを買いました。<br>      Massdropで探しても良かったのですが、なかなかほしいものが無く。国内でサクッと配送いただけて助かりました。値段もお手頃。<br>      配色、Google      SpleadSheet方眼紙で色々試して白オレンジ水色を選択したのですが、意図せずゆるキャン△色になってました。<br>      そのうちDistorteD配色とかLove Masic配色とかやってみたいです。    </p>    <h3>キースイッチ</h3>    <p>      最初はGateronの黄軸が気になっていたのですが、色々漁ってたらAliExpressに<a href="https://ja.aliexpress.com/item/Gateron-5-mechaniacl-mx/32836368723.html" target="_blank">茶色の静音軸</a>が売られていたのでポチりました。<br>      初めて海外から物発送してもらったけど、送料かかってないし普通に届いてびっくりしました。<br>      しかも65個3000円。結構安い。    </p>    <h3>道具一式</h3>    <ul>      <li>はんだごて(こて台含む)</li>      <li>テスター</li>      <li>半田吸い取り線</li>      <li>半田吸い取り機</li>    </ul>    <p>      島忠で買いました。<br>      なんだかんだ組んでるとテスターには滅茶苦茶お世話になりました。大事。    </p>    <h2>組んだ</h2>    <p>      とりあえず<a キーボードのファームウェアはqmk_firmwareで実装され、キーマップなどを自由に変更することが出来ます。      ビルドもdocker imageがあって、一発で済みます。すごい。    </p>    <p>デフォルトのキー配列からの変更点は↓のような感じで落ち着いています。</p>    <ul>      <li>        JP配列として使いたかったので<code>keymap_jp.h</code>をincludeしてJP記号のキーマップ設定      </li>      <li>        デフォルトのENTERキーの位置に<code>SFT_T(KC_ENT)</code>を設定して単発でEnter,同時押しの場合にShiftが押されるように      </li>      <li>        b押そうとしてPro        microの下のキー押ささることが多かったのでそこもbにしちゃう      </li>    </ul>    <p>      最初はマウスキーも実装してみてたんだけど、弄ってるうちに容量足りなくなって諦め…。    </p>    <p>      <a href="https://github.com/ikaruga777/qmk_firmware/blob/master/keyboards/helix/rev2/keymaps/shimarin/keymap.c" target="_blank">https://github.com/ikaruga777/qmk_firmware/blob/master/keyboards/helix/rev2/keymaps/shimarin/keymap.c</a>    </p>    <p>雑にやってるのでcommit logがマジで汚い。</p>    <h2>使ってみた</h2>    <p>作って2週間経った現状の感想ですが</p>    <ul>      <li>最高に愛着が沸く。</li>      <li>格子配列自体は慣れた。</li>      <li>space,mod,変換無変換の誤爆はまだ多め。</li>      <li>        矢印キーの誤爆多数なので、Fn同時押しにして上下二行配列にしてもいい気がした。      </li>      <li>        静音茶軸があまりにも静かすぎてドン引きする。        <ul>          <li>マジで最高。下手なメンブレンよりも静か。</li>        </ul>      </li>      <li>minila airを完全に捨て去ることはできなかった。(PUBG)</li>      <li>隣の芝(ロープロスイッチ、LED実装)は青い。</li>      <li>頼んでよかった。作ってる間楽しかったし、弄ってる今も超楽しい。</li>    </ul>    <p>と言った感じです。</p>    <h2>これからやりたい</h2>    <ul>      <li>いい感じの傾斜をつけたい。</li>      <li>OLEDのフォント変更</li>      <li>        アクリルでもう一枚作りたい(沼)。        <ul>
